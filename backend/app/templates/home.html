Todo API Documentation

A simple REST API built with Django REST Framework & JWT Authentication

Authentication

POST /api/signup/

Register a new user with the following fields:

{ "first_name": "Elyas", "last_name": "Tadesse", "username": "elyas", "email": "elyas@gmail.com", "password": "yourpassword" }

Note: username and password are required fields.

POST /api/token/

Obtain access and refresh tokens using valid credentials.

{ "username": "elyas", "password": "yourpassword" }

POST /api/token/refresh/

Get a new access token using a valid refresh token.

{ "refresh": "your_refresh_token_here" }

Todos

GET / POST /api/todos/

Retrieve all todos for the authenticated user or create a new one.

POST body example:

{ "title": "Projects", "content": "Build many projects using DRF." }

GET / PUT / DELETE /api/todos/<int:pk>/

Retrieve, update, or delete a specific todo by its ID.
